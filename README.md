# Is Today The Day That The Birth Of Jesus Is Celebrated

A module that does one thing, tells you if today is the day that people all over the world celebrate the birth of jesus.

## Use 

```js
var isTodayTheDayThatTheBirthOfJesusIsCelebrated = require("is-today-the-day-that-the-birth-of-jesus-is-celebrated");
console.log(isTodayTheDayThatTheBirthOfJesusIsCelebrated()); // true on Christmas
```
